# 🌐 Koki's Portfolio Website

このリポジトリは、私（Koki Hashimoto）が作成した**モダンでスタイリッシュなポートフォリオサイト**です。自己紹介、趣味、過去の作品を魅力的に紹介し、個人ブランディングを効果的に行うことを目的としています。

🔗 **Live Site**: [https://kokisuke.github.io/](https://kokisuke.github.io/)

---

## ✨ サイト概要

- **フレームワーク不使用**: 純粋なHTML / CSS / JavaScriptで構築
- **モダンデザイン**: 白背景＋ネオンアクセントのクリーンで洗練された外観
- **レスポンシブ**: デスクトップ、タブレット、スマートフォン全対応
- **多言語対応**: 日本語/英語の動的言語切り替え機能
- **アニメーション**: スムーズなフェードイン・ホバーエフェクトで上質なUX

---

| **Home** (`index.html`)     | ウェルカムページ。サイト全体のナビゲーションハブ |
